# Creed (2.0.0-alpha)

Creed is a forge 1.8.9 mod, Creed goal is to automate simple skyblock annoying and repetitive tasks. You can access the mod through OneConfig or /Creedmod

## Features

- Render 
  - Watermark
- Player
  - Auto Harp
  - Auto Experiments
- Other
  - DevMode (Debug)
  - Mod Hider (Enabled by default)
